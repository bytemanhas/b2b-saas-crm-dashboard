# CRM Sales Pipeline Analytics Dashboard

### SQL + Power BI | B2B SaaS CRM Analytics

## Project Overview

This project is an end-to-end CRM analytics solution built using SQL and Power BI to analyze sales pipeline performance, customer behavior, lead conversion, and opportunity management for a B2B SaaS business.

The objective of this project is to transform raw CRM data into actionable business insights that help stakeholders monitor pipeline health, identify high-performing lead sources, optimize customer targeting, and improve revenue generation.

---

## Business Problem

Sales teams often struggle to track:

* Lead generation performance
* Customer segmentation
* Opportunity conversion
* Pipeline health
* Revenue performance


---

## Database Schema

The project follows a relational CRM data model with 5 tables:

* Customers
* Leads
* Contacts
* Opportunities
* Activities

### Relationships

* Customers → Leads
* Customers → Contacts
* Customers → Opportunities
* Leads → Activities

---

## Dataset Size

* Customers → 500 rows
* Leads → 5000 rows
* Contacts → 2000 rows
* Opportunities → 3000 rows
* Activities → 12000 rows

## Dashboard Pages

### 1. Executive Overview

Provides a high-level summary of sales pipeline performance.

Key Insights:

* Total pipeline value
* Won revenue
* Win rate
* Monthly pipeline trend
* Lead source performance

---

### 2. Customer Analytics

Analyzes customer distribution, revenue contribution, and industry segmentation.

Key Insights:

* Revenue by industry
* Customer distribution
* Top customers by revenue
* Pipeline by industry

---

### 3. Lead Analytics

Tracks lead funnel and source effectiveness.

Key Insights:

* Lead funnel
* Leads by source
* Qualified leads by source
* Lead conversion analysis

---

### 4. Opportunity Analytics

Analyzes deal stages and pipeline performance.

Key Insights:

* Opportunity stage distribution
* Deal size analysis
* Pipeline by industry
* Revenue insights
